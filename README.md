# Rock Paper Scissors Game

## Overview

This project is a simple and interactive Rock Paper Scissors game developed using HTML, CSS, and JavaScript. The player competes against the computer, which generates random choices. The game determines the winner based on the traditional Rock-Paper-Scissors rules and updates the score dynamically.

## Features

* Interactive user interface
* Random computer-generated moves
* Real-time winner calculation
* Dynamic score tracking
* Responsive and attractive design
* Reset game functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* If both choices are the same, the round ends in a draw
